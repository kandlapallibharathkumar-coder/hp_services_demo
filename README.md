# Dell Authorized Service Center – Hebbal, Bengaluru

A premium, modern, fully responsive website built using only **HTML5, CSS3, and Vanilla JavaScript (ES6+)**. The design captures Dell's brand identity with glassmorphism overlays, custom vector icon visuals, responsive grid layouts, and interactive components.

---

## 📁 Folder Structure

```text
dell-authorized-service-center/
├── index.html          # Homepage (Hero, Why Choose Us, Process, FAQs, Slider)
├── services.html       # Dynamic Catalog with Instant Search & Category Filters
├── about.html          # Mission, Core Values, Milestone Timeline & Expert Crew
├── location.html       # Location Details & OpenStreetMap Iframe Map Embed
├── contact.html        # Detailed Repair Appointment Booking Form
├── css/
│   ├── style.css       # Core Layouts, Design System Tokens & Dark Theme Variables
│   ├── responsive.css  # Viewport Media Queries (Desktop, Laptop, Tablet, Mobile)
│   └── animations.css  # Custom Keyframes, Scroll-reveal triggers & Button Ripples
├── js/
│   ├── script.js       # Core Actions: Navbar, Theme Toggles, Modals, Sliders, Counters
│   ├── services.js     # Data Repository of 21 Services with Search/Filters Logic
│   └── contact.js      # Form Validation Constraints & URL Query Prefill Helper
└── README.md           # Documentation File
```

---

## 🌟 Key Features

1. **Brand Identity Styling**: Styled using premium Gold and Bronze colors (`#C5A059`, `#8E6F3E`, `#E5C180`) with harmonious font hierarchies using Google Font Poppins.
2. **Dynamic Service Catalog**: Custom lists of **21 specialized Dell services** complete with timeframes, starting price estimates, and dynamic filtering.
3. **Smooth Interactive Transitions**: Includes custom typewriter headings, count-up numeric counters, click ripple indicators, and active accordion FAQ slides.
4. **Adaptive Dark Mode**: Toggle states persist across pages using client-side `localStorage`.
5. **Robust Validation Form**: The contact submission filters dates, Closed Weekends, and incorrect phone/email strings before dispatching.
6. **No External Libraries**: Completely independent of jQuery, Bootstrap, or Tailwind to load instantly on slow mobile connections.

---

## 🚀 How to Run

1. Extract the project ZIP package.
2. Double-click or open **`index.html`** in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Navigate between pages using the sticky header bar.
